# Mobile SOC Lab

A portable cybersecurity laboratory built
to run enirely on a mobile device using
Termux and Kali NetHunter

## Overview
This repository documents the construction
of a mobile Security Operations Center
toolkit capable of performing real world
security investigations from a smartphone.

## Projects
01. Pocket SOC Toolkit
    Network reconnaissance, vulnerability
    discovery, and automated reporting

02. Network Recon (coming soon)

03. Vulnerability Scanner (coming soon)

04. Metasploit Playbooks (coming soon)

## Skills Demonstrated
- Network Reconnaissance
- Service Enumeration
- Vulnerability Assessment
- Security Automation
- Python Scripting
- Exploitation Workflows
- Incident Investigation

## Tools Used
- Nmap
- Python 3
- Metasploit Framework
- Kali NetHunter
- Termux

- Platform: VMWare Workstation Pro (Bridged)
